(The beginnings of) a 2D game engine written with C#/.NET.  
I'd like to complete it at some point, but it's been on the backburner for a while, and likely will continue to be for some time.  
  
## Requirements
- Gtk#
- Gdk
- GLib#
- Atk#
- Pango
- Gtk .NET
- Mono.Cairo
- Box2DX
- Tao.OpenGL (?)
- Tao.Platform.Windows (?)
